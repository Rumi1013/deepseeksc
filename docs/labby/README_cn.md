# [Labby by LabEx](https://labex.io/)

LabEx 是一个专注于编码和技术的互动式实践学习平台。它结合了动手实验、人工智能辅助和虚拟机，提供无视频的实践学习体验。

Labby 是一个由 DeepSeek R1 驱动的学习助手，支持学生练习代码的自动验证，并提供即时反馈、调试和代码生成。
